# 2018年 北港朝天宮 贊境 士林葫蘆寺 建廟70週年遶境

台北士林葫蘆寺慶祝建廟70週年祈安遶境大典，北港朝天宮慶贊遶境！

每場活動都是各位的付出，所以感謝各位工作人員、感謝各轎班人員、感謝炮場的工作人員、感謝攝影組的夥伴，感謝大家！

照片中若有各位的親朋好友，歡迎大家幫忙標註一下吧！
